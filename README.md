```
   git config --globla user.email "your email"
```

## Branching Strategy

 - Create a branch
 ``` 
 git checkout -b feature-branch 
 ```
 - Merge a branch
 ```
 git checkout main
 git merge feature-branch
 ```
 - Delete a branch 
 ```
 git branch -d feature-branch
 ```
